Quarterly Profit Analyzer
Author : Laura Romero


Java Application tool for small business owners to analyze quarterly financial performance and make informed expansion decisions.


Over view :
This will help business owners :
*Calculate net profit
*Calculate net profit margins
*Check business profitability levels
*Plan expansion budgets and maintaining adequate liquidity


Features
*Profit Analysis: Automatically calculates net profit and profit margins from revenue and expenses data
*Profitability Assessment: Provides context feedback on financial decisions.
*Comprehensive Reporting: Generates detailed quarterly summaries.

Structure
src/
├── ProfitCalculator.java          # Core business logic and calculations
├── QuarterlyReport.java            # Data model for quarterly financials
├── QuarterlyProfitAnalyzer.java    # Main application and user interface
└── test/
    ├── ProfitCalculatorTest.java   # Unit tests for business logic
    └── QuarterlyReportTest.java    # Unit tests for data model

Pre requesites
*Java Development Kit (JDK) 11 or Higher
*JUnit 5 (Jupiter) for running tests


How to run :

Compile
javac *.java

Run the application
java QuarterlyProfitAnalyzer
